# Integrate Jaeger in ASP.NET Core

<img src="https://www.yogihosting.com/wp-content/uploads/2026/03/jaeger-aspnet.png" alt="Integrate Jaeger in ASP.NET Core"  title="Integrate Jaeger in ASP.NET Core">

Here we will integrate Jaeger to capture traces of ASP.NET Core app. The complete tutorial can be read from - <a href="https://www.yogihosting.com/aspnet-core-opentelemetry-Jaeger/" target="_blank">How to integrate OpenTelemetry with Jaeger for  Distributed Tracing in ASP.NET Core</a>.

## What is covered

- [x] SQL Server and Entity Framework Core Traces
- [x] HTTP Web API Traces
- [x] MassTransit and RabbitMQ Traces
- [x] Redis Cache Traces
- [x] Custom Traces with ActivitySource class

## How to run the app

Make sure you run:
1. EF Core migrations which will create the database.
2. Run Jaeger on your PC
3. Run Redis Cache from Docker Container
4. Run RabbitMQ from Docker Container
